branch html -branch: bifurkasi dari kondisi kode yang membuat alur baru. -pull request: istilah yang bisa kita artikan sebagai permintaan untuk menggabungkan kode. -merge: penggabungan antara camang 1 dengan cabang lainnya -conflict: konflik yang terdapat di github -push: menyimpan perubahan yang telah kita lakukan ke dalam sebuah repository yang berada di git server -pull: menarik seluruh data perubahan yang telah dilakukan apabila kita tertinggal beberapa titik dibawah. -fetch: mengambil perubahan tanpa merging kedalam branch yang aktif. -remote: tempat penyimpanan di server git. -commit: menyimpan perubahan kita di local baru melakukan push ke server git.
